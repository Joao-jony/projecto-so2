Simulação de Sistema Operativo - UNITEL

Descrição
Simulação de um sistema operativo para gerenciamento de vendas de cartões SIM da UNITEL em ambiente multiprocessador.

Objetivos
- Implementar sincronização entre processos/threads
- Gerenciar prioridades de vendas (empresa vs público)
- Simular interrupções e alocação dinâmica de memória
